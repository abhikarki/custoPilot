Embed Code Example

To embed this widget on any website, use the following code:

<script>
  window.CustoPilotConfig = {
    chatbotId: "YOUR_CHATBOT_ID"
  };
</script>
<script src="http://localhost:8080/static/widget.js" async></script>

How It Works

  • Session-Based: Each page visit creates a unique conversation session
  • No Login Required: Works for anonymous website visitors
  • AI-Powered: Responses come from your knowledge base via AI
  • Customizable: Widget colors match your chatbot settings
